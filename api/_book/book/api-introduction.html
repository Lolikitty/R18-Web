簡介 : 關於矽桶

說明 : API

本 API 以 REST-API (JSON) 與 矽膠桶 做溝通

本 API 全部用小寫表示；兩個字母(數字)之間，使用底線 ( _ ) 區隔

API 中的縮寫：

加速度計 (Accelerometer) : acc
IC溫度計 (IC-Temperature) : ic_temp
按鈕 1 (Button-1) : btn_1
按鈕 2 (Button-2) : btn_2
LP 按鈕 1 (LP-Button-1) : lp_btn-1
LP 按鈕 2 (LP-Button-2) : lp_btn-2
